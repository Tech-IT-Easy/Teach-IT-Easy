Module games.Progg.RightMenu

This class contains methods for interacting with the right-hand menu

Functions

new (maxCommands) Creates the Right-hand screen in the programming game
load () Load function
show (bottomInput, preventFalse) Shows all elements in the menu Currently also used to test certain commands
highlight (command) Highlights the command in the specified position.
removeHighlight (command) Removes the highlight on the command in the specified position.
play () Changes the play-button to a stop-button when the play-button is pressed Also removes any highlight on the commands
stop () Changes the stop-button to a play-button when the stop-button is pressed
standardLayout () Called to write the standard layout
loopLayout () Called to replace buttons for loops
buildLayout (bottomInput) Called to replace buttons for use in the build area
ifTrueLayout (preventFalse) Called to replace buttons for use with if-statements
optionsLayout () Called to replace buttons for options when selecting an action
moveLayout () Called to change button layout to show information message for user when moving an action
confirmLayout () Called to changed button layout to show confirmation window for clear all.
rightMenuEventHandler:update (object:RightMenu, eventListener:EventListener, event:Event) Handles all input to the rightMenu.
