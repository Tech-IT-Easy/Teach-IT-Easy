Module games.Progg.DrawRightMenu

This class keeps contains functions to draw elements in the right-hand menu

Functions

new (maxCommands) Creates the draw functions for the right menu
getPosition (command) Assigns commands to numbers
getFileName (action) Gets the file name for a specified command
drawHighlight (command) Draws a highlighted box for the specified command
drawRemoveHighlight (command) Draws a un-highlighted box for the specified command
drawRow (row) Draws a row of 3 command-boxes
drawTwoBoxRow (row, r, g, b) Draws two boxes covering a row Filling color is specified in rgb
drawFullRow (row, r, g, b) Draws a wide box covering a full row Filling color is specified in rgb
addImages () Adds images for all commands to the 9-command layout
addImage (command) Adds a image for a specified command in the 9-command layout
addNumbers () Adds all numbers to the 9-command layout
addSingleNumber (position) Adds a single numbers to the 9-command layout
addPlay () Adds the play text to a single wide button in the 9-command layout
addStop () Adds the stop text to a single wide button in the 9-command layout
addBack () Adds the play & back text to the two-buttons row in the 9-command layout
addIfFalse (preventFalse) Adds if false text to the two-buttons row in the 9-command layout.
addOptions (can_enter) Adds the option buttons when selecting an action
addMoveInstructions () Draws an instruction message in the right menu for moving an action.
addLoopOptions () Adds numbers to the loop layout
preventRecursion (allow_if) Greys out unusable commands
